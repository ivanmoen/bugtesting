3 · Projects

SVN URL, credentials, and Steam settings live on the panel — this worker only needs to know where each project's .uproject file lives locally.

4 · Credentials (optional)

Saved DPAPI-sealed at %ProgramData%\RPGBuildServer\credentials.dat (machine-scope; service can read).

Steam's password flow goes through a cached Steam Guard sentry — only the username is stored. Run steamcmd +login <user> once on the build PC to seed the sentry.
